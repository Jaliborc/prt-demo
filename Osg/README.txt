Main Files
Glove - reacts to MeasureHand if the application is running.
Recording - loads a previously recorded animation.
Pdt - tests the pdt algorithm on static geometry.
Pgt - loads a binary pgt file (deprecated).

Commands:
Up/Down Arrows - change environment
Left/Right Arrows - move hand
Q/A - increase/reduce diffuse
W/S - increase/reduce specular
