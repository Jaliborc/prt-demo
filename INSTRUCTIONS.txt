## How to Run
1. Go to "Computer -> Proprieties -> Advanced System Settings" and append to the environment variable PATH:
	<this folder>\Osg\Dependencies\osg-3.1.3\bin;

2. Type in the command line:
	cd <this folder>\Osg
	..\Release\OsgViewer
	
## How to Use
Up/Down Arrows - change environment
Left/Right Arrows - rotate hand
Q/A - increase/reduce diffuse
W/S - increase/reduce specular
